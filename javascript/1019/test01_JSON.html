JSON(JavaScript Object Notation)

서버와 클라이언트간의 자료전달방식 : XML,JSON
        모바일(웹)에서의 서버와 데이터 전송방식에서 사용
        프로그램의 종류나 플랫폼에 상관없이 사용할 수 있다.(예:java,C,C++,Python,PHP,코틀린,GO언어 등);
        자바스크립트의 객체와 같이, "키(key):값(value)"로 구성되어있다.
        처리방식은 Object를 String 으로 직렬화(serialize) 처리를 해서 전송하며,
        다시 사용하고자 한다면 Json문서형식을 Javascript Object형식으로 변환하여 사용해야 한다.

        Object ↔ JSON
        형식으로 서로 변환해야함
