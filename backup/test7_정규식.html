정규식 (RegularExpression)

문자열 : 

            형식 : /~~/플러그인,  /[~~]/플러그인        자바:: ^/~~~?+$
            플러그인 :
            g(global) : 전체범위에서의 처리
            m(multiple) : 여러줄에 걸쳐서 처리 (textArea 태그)
            s(single line) : 한줄에서의 처리 (기본값으로 생략함)
            i : 영문자인 경우 대/소문자 구분을 하지 않는다
            u : 유니코드에 대한 구별 처리요

            정규식을 실행하기 위해서 사용되는 필수 요소
            1.사용자 입력 문자열.match (정규표현식 or 정규표현식변수) : 포함 유무에 대한 비교시에 사용하면 편함
            2.정규 표현식 변수.test(사용자입력문자열) : 구체적인 문(열)간의 비교체크애 사용시 편히

Front에서 값의 전송(form태그 사용)시는 사용자 값을 입력 후 반드시 유효성 검사**필수**
